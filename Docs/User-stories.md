# User Stories

## Authentication

As a user, I want to create an account so I can save my progress.

As a user, I want to log in securely so I can access my data.

## Resume Analysis

As a user, I want to upload my resume so I can receive feedback.

As a user, I want to see a resume score so I can understand where I need improvement.

## Interview Practice

As a user, I want AI-generated interview questions so I can practice.

As a user, I want feedback on my answers so I can improve.

## Dashboard

As a user, I want to track my progress so I can measure improvement.
